# AI Mathanda - Solusi Matematika Teknik

## Project Overview
AI Mathanda is a web application designed to assist users in solving technical mathematics problems using AI technology. The platform provides step-by-step explanations for each solution, allowing users to learn and solve problems more effectively. Users can also upload images of their math questions and receive instant solutions. 

## Installation
To set up the project locally, follow these steps:

1. **Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/ai-mathanda.git
   cd ai-mathanda
   ```

2. **Open in a Web Server**
   - This project uses HTML, CSS, and JavaScript. You can run it by opening the `index.html` file directly in your web browser, or better yet, you can use a local web server like [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in Visual Studio Code for live-reloading capabilities.

## Usage
1. Open your browser and navigate to `index.html`.
2. Use the provided navigation to log in to the application via `login.html`.
3. Once logged in, you’ll have access to the dashboard (`dashboard.html`), where you can interact with the AI, upload math questions, and manage token purchases.

## Features
- **AI Assistance:** Get help from an intelligent AI system for your math problems.
- **Detailed Explanations:** Step-by-step guides on solving each problem.
- **Image Upload:** Ability to upload photos of math problems for instant solutions.
- **User Management:** Login functionality and user profiles.
- **Token Management:** Purchase token packages for interaction with the AI.

## Dependencies
The project utilizes the following external libraries:
- **Tailwind CSS:** A utility-first CSS framework for styling.
- **Font Awesome:** For iconography and design elements.
- **Google Fonts (Inter):** For a modern font display.

As per the provided `package.json`, there are no additional backend dependencies specified at this time.

## Project Structure
The project is organized as follows:

```
ai-mathanda/
│
├── index.html        # The main landing page
├── login.html        # User login page
├── dashboard.html     # Main application interface after user login
└── js/               # Directory for JavaScript files
```

Each HTML file is structured to provide the necessary UI components using Tailwind CSS for styling, with JavaScript implemented for interactivity.
